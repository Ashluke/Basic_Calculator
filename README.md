# Basic Calculator (Java)
A simple calculator built using Java Swing. Performs basic arithmetic operations: addition, subtraction, multiplication, and division via a graphical user interface (GUI).

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Simple and user-friendly GUI

## Project Structure
Calculator/
├── src/
│   └── Main.java
├── README.md
└── .gitignore

## How to Run
1. Ensure Java is installed.
2. Open a terminal in the project folder.
3. Compile the Java file:
   ```bash
   javac src/Main.java
4. Run the program:
    java -cp src Main
